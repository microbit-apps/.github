## VS Code extension for MakeCode

To use VS Code to create micro:bit apps, do the following:

1.  Install VS Code
2.	Clone https://github.com/microsoft/vscode-makecode
3.	cd to vscode-makecode
4.	yarn install
5.  code .
6.	F5 to run new instance of VS Code
7.	In this new instance, open a folder containing a micro:bit app, such as
     - https://github.com/microbit-apps/microdata
8.  Click on the MakeCode icon on the left of VS Code to see options
